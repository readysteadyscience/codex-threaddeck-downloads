# ThreadDeck v0.2.0

ThreadDeck v0.2.0 is a public-safe control-plane package for Codex execution routing, controller coordination, and worker collaboration.

## Highlights

- Installs a small project kit containing `AGENTS.md` and `.threaddeck/`.
- Adds local CTD state files for registry, status, tasks, handoffs, and artifacts.
- Standardizes visible conversation grammar: `⇄` for Controller, `↳` for Worker or Specialist.
- Keeps normal usage simple: install CTD, continue working in Codex, and let CTD recommend the smallest useful execution surface.
- Treats visible worker conversations as persistent context lanes, not as a fixed default Docs/Tests/Implementation/Release set.
- Bundles a preview execution-surface recommender for current conversation, Codex native subagents, visible workers, or manual TaskCards.
- Bundles an advisory auto-route lifecycle hook preview for recording CTD routing decisions after hook trust review.
- Bundles a dispatch-plan preview that turns the latest routing decision plus the local registry into a safe controller plan before any thread tools are used.
- Keeps real cross-thread dispatch bounded by the Codex thread tools actually available in the current conversation.

## Packages

- `ctd-project-kit-v0.2.0.zip`: installable project kit and local installer.
- `ctd-plugin-v0.2.0.zip`: experimental plugin preview skeleton. It is included for product planning and beta testing only.

## Boundaries

- This release is not a source release.
- CTD does not bypass Codex, OpenAI, filesystem, account, or project permissions.
- CTD does not guarantee that every Codex conversation exposes thread tools.
- The hook preview does not rewrite prompts, create threads, or dispatch messages.
- The dispatch planner does not create threads or send messages.
- High-risk actions still require explicit user confirmation.

## Upgrade Note

For existing projects, keep a copy of the current `.threaddeck/` directory before replacing it manually. The installer refuses to overwrite an existing `.threaddeck/` directory.
